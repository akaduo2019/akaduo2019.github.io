类的继承

创建子类时，父类必须包含在当前文件中，且位于子类前面。
定义子类时，必须在括号内指定父类名称。
super()是一个特殊函数，帮助Python将父类和子类关联起来：这行代码让Python调用父类的方法init(),让子类实例包含父类的所有属性。
